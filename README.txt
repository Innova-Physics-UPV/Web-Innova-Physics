Carpeta destinada al desarrollo y organización de la nueva Landing Page de Innova Physics UPV.

Los requisitos mínimos de esta Landing Page son:
    - Mostrar todo el contenido representativo del equipo: (objetivo, miembros, sponsors)
    - Página web responsiva, adaptable a cualquier formato
    - Código limpio, fácil de leer y de actualizar, la página deberá ser adaptable para futuros cambios hechos por 
      distintos miembros del equipo.

Ideas a tener en cuenta:
    - Integraciones con redes (API, embedd...) de modo que el contenido de la web se actualice con nuestro contenido en redes
    - Integración con CERN Courier, noticias de aceleradores...
    - Idea a largo plazo: API que facilite el acceso a datos de EXFOR
    - Acceso sencillo a forms y redes

Idea de diseño:
    - Página de inicio con introducción a apartados
    - Página dedicada a miembros
    - Página dedicada a explicación técnica (BNCT, ciclotrón...)
    - Página dedicada a competición
    - Página dedicada a sponsors

Posibles herramientas para usar:
    - Astrojs
    - Typescript
    - React
    - Flask
    - Python
